在 git commit 说明中也可以通过特殊字符来转义一些小表情

```shell
git commit -m "fix：:bug: 修复一个缺陷"
```

这是别人看到的 commit 信息 => fix：:bug:修复一个缺陷


🎨 - `:art:` - 改进代码的结构/格式.  
⚡️ - `:zap:` - 提高性能.  
🔥 - `:fire:` - 删除代码或文件.  
🐛 - `:bug:` - 修复一个缺陷.  
🚑 - `:ambulance:` - 关键的热修复补丁.
